# Professional Portfolio Card - Luis Enrique León Bolado
[Preview de la Tarjeta](preview.jpg)

Este repositorio contiene el código fuente de mi tarjeta de presentación profesional interactiva. El objetivo de este proyecto es demostrar habilidades en **Frontend Development**  aprednidas en la certificacion de Meta de Full-Stack (HTML5/CSS3) y consolidar mi marca personal como **Backend Developer**.

**Live Demo:** [https://enrique1103.github.io/presentacion_en_proceso/](https://enrique1103.github.io/presentacion_en_proceso/)

---

## Objetivo del Proyecto
Crear una interfaz minimalista, responsiva y de alto impacto que resuma mi perfil híbrido entre la **Ingeniería Eléctrica** y el **Desarrollo de Software**. La estructura está inspirada en principios de diseño limpio para facilitar la lectura por parte de reclutadores y equipos técnicos.

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica para SEO y accesibilidad.
* **CSS3 Moderno:** * **Flexbox:** Para la disposición dinámica de elementos.
    * **Glassmorphism:** Efectos de desenfoque y transparencia en el formulario.
    * **Animaciones:** Micro-interacciones y transiciones en los estados de `hover`.
    * **Responsive Design:** Optimización total para dispositivos móviles mediante Media Queries.
* **GitHub Pages:** Hosting y despliegue automatizado.
